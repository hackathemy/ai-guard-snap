## API Reference

### Send message

**Send message for about smart contract security and explanation via chatgpt assistant api**

**POST** `http://api.guard.ai/message`

**Request body**

- `chainId` (string) - **Required**  
  The unique identifier for the blockchain network.

- `data` (object) - **Required**  
  The data object containing details about the transaction.
  - `method` (string) - **Required**  
    The blockchain method being called, such as `eth_sendTransaction`.
  - `params` (array) - **Required**  
    An array of objects containing transaction parameters.
    - `from` (string) - **Required**  
      The address of the account sending the transaction.
    - `to` (string) - **Required**  
      The recipient address of the transaction.

### Example Call By Nodejs

```js
const axios = require("axios");

const callData = {
  chainId: "1261120",
  data: {
    method: "eth_sendTransaction",
    params: [
      {
        from: "0xb0b9c5F027A59409579A6a9139c4E9BB29De5A4b",
        to: "0xdC0e4E9C1BF3Aa88Cd9BE32186a741cd893C78cA",
        value: 1000000000000,
        gasLimit: "0x5028",
        maxPriorityFeePerGas: "0x3b9aca00",
        maxFeePerGas: "0x2540be400",
      },
    ],
  },
};

const axiosInstance = axios.create({
  baseURL: "http://localhost:3000/",
  headers: {
    "Content-Type": "application/json",
  },
});

const result = await axiosInstance.post("/message", callData);
```
